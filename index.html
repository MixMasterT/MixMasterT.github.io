Fractalscope

Vanilla JavaScript (ES6) project that leverages the Canvas API to render beautiful images

  • Rapidly determines pixel colors by recursively calculating the function f(x) = x^2 + C for values in the complex plane, rendering the famous Mandlebrot Set as a result
  • JavaScript key binding gives users hands-on zoom and pan control
  • ‘Rotate colors’ feature animates the color scheme and brings fractal to life

This Mandlebrot Set viewer was built in one week in vanilla JavaScript (ES6 with Webpack), html and css. It features controls to zoom in/out and pan around the Complex plane decorated with an image of the fractal. Users can adjust the maximum number of iterations through which the image is generated. Color rotation is also enabled to bring the image to life.

About

The one thing that has remained constant in my life is a desire to constantly learn and grow. From a young age I always wanted to travel the world and learn about how people live in other countries. As an undergrad, I studied Linguistics and traveled to China. That eye-opening experience inspired me to move to Taiwan, and become immersed in Chinese culture. I earned a Master's degree while in Taiwan, and realized that as much as I love the life I had been living there, I was ready for something new. In the process of writing my thesis, I discovered that I really enjoyed computer programming, and started learning what I could about various web technologies online.

Eventually, I decided that guidance from a knowledgeable expert would allow me to learn faster, and with greater confidence, so I applied and got in to App Academy. I am thrilled to have had the opportunity to learn among such a talented cohort of fellow students. As a recent graduate, I am now ready to pursue full-time employment as a software engineer. I look forward to putting my newly honed skills to use for the right team.
